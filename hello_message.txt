# Super User priviledges:

Note that some things (like running the setup script which sets up the can device) require super user priviledges.

A convenient way to set this up to run with elevated priviledges when you have no keyboard is to create a script where we run these sudo commands and add it to the sudoers list.

Here is how:

**Modifying the sudoers file:**

To allow your command to run without requiring a password, you can modify the sudoers file:

1. Open a terminal and type `sudo visudo`. This will open the sudoers file in a safe way, preventing multiple simultaneous edits and doing syntax checking on the changes.

2. At the bottom of the file, add a line like this:

    ```
    yourusername ALL=NOPASSWD: /path/to/your/command
    ```

    Replace `yourusername` with your actual username, and replace `/path/to/your/command` with the full path to the command or script you want to run. This line allows your user account to run the specified command with sudo, without needing to enter a password.

3. Save and exit the file. If you're using the default editor (nano), you can do this by pressing `CTRL + X`, then `Y`, then `Enter`.

Then, in your startup applications command, use `sudo /path/to/your/command`.
